android-login-google-plus
=========================

This repo has the code for this tutorial I am following: http://www.androidhive.info/2014/02/android-login-with-google-plus-account-1/

El servicio Google+ Sign-in permite a los usuarios acceder a tu aplicación Android con los credenciales de Google. Así una vez se logean también estarán disponibles datos como su nombre, email, fotos de su perfil, ect. No solo se puede hacer login sino que también se pueden hacer otras cosas como publicar en su cuenta de Google Plus, obtener la lista de personas y sus círculos, etc.
